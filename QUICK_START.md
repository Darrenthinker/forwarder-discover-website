# 🚀 快速启动指南

## 一键启动 (推荐)

```bash
node start.js
```

## 其他启动方式

```bash
# 方式2: Bun脚本
bun auto-start.js

# 方式3: Shell脚本
./.same/start.sh

# 方式4: 传统方式
bun run dev
```

## 访问地址

启动成功后，在浏览器访问：
**http://localhost:3000**
